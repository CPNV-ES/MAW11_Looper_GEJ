## Getting Started

### Prerequisites

* Docker 4.33.0

### Configuration

Make `.env` file 
and put the 
```env
POSTGRES_USER=admin
POSTGRES_PASSWORD=mysecretpassword
PGADMIN_DEFAULT_EMAIL=admin@example.com
PGADMIN_DEFAULT_PASSWORD=admin
```
or copy `.env.example`
